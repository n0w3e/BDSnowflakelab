# Лабораторная работа №1 - Нормализация данных в схему "снежинка"

## Задание

Трансформация данных о продажах товаров для домашних питомцев из CSV файлов в нормализованную схему "снежинка" в PostgreSQL.

## Запуск

```
docker-compose up -d
```
## Подключение через DBeaver
```
Host: localhost
Port: 5432
Database: POSTGRES_DB
User: POSTGRES_USER
Password: POSTGRES_PASSWORD
```
